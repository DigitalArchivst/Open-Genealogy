# Records Folder

Put copies of record images, downloads, transcriptions, abstracts, and source excerpts here.

Do not put your only copy of anything in this folder.

## Privacy Note

Codex, Claude Code, and Cowork can read local files, but their standard modes process file contents with cloud models. Keep out documents too sensitive for a cloud service. For highly sensitive work, use a fully local setup or remove sensitive details before analysis.

Protect living persons. Do not add files with addresses, contact details, health information, financial information, or other sensitive details about anyone who could plausibly be alive unless you have deliberately chosen an appropriate privacy-safe workflow.
