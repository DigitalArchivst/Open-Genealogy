# Tool Ladder

Use this file to understand where this workspace fits.

```text
chat
  -> Custom GPTs / Gems
  -> Projects in ChatGPT or Claude
  -> Claude Cowork
  -> Codex / Claude Code
  -> LM Studio for fully local work
```

## Three Meanings Of Local

| Meaning | What It Means | Examples |
| --- | --- | --- |
| Local files | Your research files live on your computer | Codex, Claude Code, Claude Cowork |
| Local workspace | The AI tool can read, organize, create, and revise files in a folder | Codex, Claude Code, Claude Cowork |
| Local model | The model itself runs on your computer | LM Studio |

Do not confuse local files with a local model. A tool can work in your folder and still use a cloud model.

## Best Fit

| Need | Best starting point |
| --- | --- |
| Quick question | ChatGPT or Claude chat |
| Reusable behavior | Custom GPT or Gem |
| Reusable context with uploaded files | ChatGPT Project or Claude Project |
| Non-programmer folder work | Claude Cowork |
| Repeatable local project workflow | Codex or Claude Code |
| Fully local privacy or offline work | LM Studio |
