# Ashe County Historical Model Specimens

> [!CAUTION]
> **These files are preserved historical model outputs, not validated research or a reproducible benchmark.** Their value is instructional: readers can inspect strengths, unsupported claims, citation practices, evidence-language errors, and uncertainty handling. No current model ranking should be inferred from this set.

## Editorial Boundary

This remediation does not rewrite the four linked output files. Current corrections and limitations are recorded on this page rather than silently inserted into the model text. Links, claims, dates, and evidence labels inside a specimen therefore reflect the historical output, not editorial verification.

The packet does not retain the exact user task, run dates, model versions and settings, source corpus, interface state, browsing or tool state, retrieval logs, criterion-level score sheets, scorer calibration, or adjudication record. Without those materials, the runs cannot be reproduced and an aggregate score cannot be audited.

## What the Writing Rubric Can Show

The [Genealogical Writing Evaluation Rubric](../../rubrics/genealogical-writing-rubric.md) can support structured discussion of what appears in the prose: source transparency, uncertainty language, conflict handling, narrative clarity, and research leads. It does not by itself establish:

- Factual accuracy, which requires independent claim-level verification
- Research exhaustiveness, which requires a defined question, search plan, corpus, and search log
- GPS alignment of a completed research conclusion, which requires evaluation of the underlying research process and evidence
- Reproducibility, which requires the missing run, retrieval, and scoring records

Each dimension should be evaluated and reported separately.

## Responsible Teaching Use

Use the files to practice annotation, identify claims that need verification, compare citation traceability, discuss evidence terminology, and design a stronger evaluation protocol. Do not use them to choose a current model or to repeat the withdrawn exact ranking.
