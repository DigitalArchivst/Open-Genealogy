# Trifecta: Complete Reference Edition

This directory contains the complete reference edition accompanying Steve Little's guided Substack article, **Trifecta: One Genealogist, Two AI Assistants, One Folder**.

## Choose Your Link

- [Read the complete reference on GitHub](trifecta-full-reference.md).
- [Open the raw Markdown for a chatbot](https://raw.githubusercontent.com/DigitalArchivst/Open-Genealogy/main/reference/trifecta/trifecta-full-reference.md).
- [Download the Lawrence First Project records package](https://github.com/DigitalArchivst/Open-Genealogy/releases/download/trifecta-lawrence-first-project-v1.0.0/trifecta-lawrence-first-project-v1.0.0.zip).

The raw Markdown URL returns the article as plain text. Give that URL to a chatbot and ask it to help you work through the guide one checkpoint at a time. Whether a particular chatbot can retrieve a URL depends on that service's tools and account settings; if it cannot, download the Markdown file and upload it directly.
